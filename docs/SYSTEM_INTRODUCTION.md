# 🚀 네이버 쇼핑 가격 추적 자동화 시스템 소개

이 문서는 사내 제품 기획 및 시장 단가 모니터링을 위해 구축된 **'가격 추적 자동화 시스템'**의 주요 기능과 활용 방안을 신규 부서 및 사용자에게 소개하기 위한 안내서입니다. 본 시스템을 도입함으로써 수작업으로 쇼핑몰을 검색하고 엑셀에 가격을 기록하던 과도한 업무 방식에서 벗어나, 24시간 자동으로 데이터를 수집하고 변동 사항을 즉각적으로 파악할 수 있게 됩니다.

---

## 1. 시스템 핵심 요약

1. **완전 자동화 (Fully Automated):** 구글 시트(Google Sheets) 데이터베이스와 깃허브(GitHub) 자동화 스케줄러(Actions)를 연동하여, 관리자의 별도 개입 없이도 정해진 시간마다 자사/경쟁사 제품의 최저가 데이터를 수집합니다.
2. **시각화 대시보드 (Dashboard):** 복잡하고 어려운 데이터베이스 직접 다운로드 없이, 직관적으로 가격 추이를 열람할 수 있는 독립된 사내 전용 웹사이트(Dashboard)를 제공합니다.
3. **쇼핑몰 집중 모니터링 (Mall Tracker):** 특정 경쟁사나 자사가 등록한 전체 상품 목록, 가격 변동 추이, 상품 추가/삭제 여부를 감시할 수 있는 리포트를 발행합니다.
4. **즉각적인 이메일 알림 (Alerting):** 타겟 상품의 가격이 하락(변동)했을 경우, 설정된 부서 및 담당자들에게 즉시 알림 메일을 단체 발송하여 빠른 시장 대응 체계를 구축합니다.

---

## 2. 주요 기능 상세 안내

### 📊 기능 1: 데이터 시각화 웹 대시보드 (Web Dashboard)

별도의 프로그램 설치 과정 없이, 팀 내부에서 공유된 URL 링크만 데스크톱 브라우저로 접속하면 실시간으로 축적된 가격 데이터를 열람할 수 있습니다.

> 📷 **[실제 대시보드 메인 화면 스크린샷 캡쳐 이미지 삽입 위치]**
> *(여기에 대시보드의 전체 요약화면이나 차트 화면을 캡쳐하여 올려주세요)*

#### 대시보드의 주요 기능
*   **전체 상품 통합 요약 (Summary Table)**
    *   현재 추적 중인 모든 타겟 상품들의 '오늘의 최저가'를 리스트 형태로 나열하여 한눈에 조망할 수 있습니다.
    *   전일 대비 가격이 하락했는지(🔹초록 화살표), 상승했는지(🔺빨간 화살표), 또는 방어 중인지(✅회색) 명확한 시각적 지표를 제공합니다.
*   **개별 항목 기간별 추이 차트 (Trend Chart)**
    *   단순히 오늘 가격이 어디가 싼지를 넘어, **시간의 흐름에 따라 해당 상품의 가격이 어떻게 변해왔는지**를 시계열 그래프로 시각화합니다.
    *   경쟁사의 프로모션 시작 시점이나 단가 인하 시기를 사후에 추적하고 패턴을 분석하는 데 유용하게 활용됩니다.
*   **최우선 판매처 직결 링크 제공 (Direct Seller Link)**
    *   비용 스캔 결과, 현시점에서 가장 공격적인 최저가를 올린 업체명(스마트스토어, 오픈마켓 등)의 이름과 이동 링크가 직접 제공됩니다. 클릭 시 해당 판매 페이지로 이동하여 실제 판매 환경을 실시간으로 모니터링할 수 있습니다.

### 📝 기능 2: 쇼핑몰(Seller) 전용 전수 추적 리포트 (Mall Tracker)

특정 단일 상품의 최저가를 찾는 것을 넘어, **"특정 타겟 업체(경쟁사, 공식 파트너사 등)가 보유한 모든 상품의 동향"**에 집중 추적하는 전용 기록 리포트입니다.

> 📷 **[실제 쇼핑몰 추적 리포트 화면 스크린샷 캡쳐 이미지 삽입 위치]**
> *(여기에 mall_report.html 실행 화면을 캡쳐하여 올려주세요)*

#### 타겟 업체 추적 리포트 주요 기능
*   **상품 라인업 노출/낙점 감시**
    *   해당 업체가 수시로 어떤 새로운 상품을 검색 결과에 노출시켰는지, 반대로 어떤 상품을 품절/삭제 처리하여 내렸는지 한눈에 파악할 수 있는 상품명 기준 이력을 제공합니다.
*   **상품별 개별 가격 변동 추적**
    *   해당 업체가 취급하는 모든 타겟 기종들에 대해 이전 가격 대비 얼마나 올랐는지, 가격을 내렸는지(상승/하락폭 표시)를 표 형태로 나열합니다.
*   **무알림 사일런트 기록 (Silent Archiving)**
    *   수십 개의 항목에 대한 가격이 바뀔 때마다 스팸성 메일이 가는 것을 방지하기 위해, 타겟 업체 추적 기능은 순수하게 DB 보관과 '조회용 리포트 생성' 목적으로만 작동하여 업무 피로도를 낮췄습니다.
    *   대시보드 상단 메뉴바의 `[📊 쇼핑몰(Seller) 추적 리포트]` 버튼을 통해 쉽게 열람 가능합니다.

### 📧 기능 3: 스마트 변동 알림 (Email Alert)

현업 담당자가 업무 중에 매번 대시보드를 새로고침하지 않아도, 중요한 가격 하락 이벤트가 감지되었을 때 시스템이 선제적으로 알려주는 스마트 알림 기능입니다.

> 📷 **[실제 이메일 수신 화면 스크린샷 캡쳐 이미지 삽입 위치]**
> *(여기에 스마트폰 메일 앱이나 PC 아웃룩으로 가격 하락 알림이 온 화면을 캡쳐하여 올려주세요)*

#### 이메일 알림의 주요 기능
*   **타겟 가격 하락 즉각 감지 시스템**
    *   자동 수집 봇이 순회를 돌던 중, 등록된 주요 모니터링 타겟 상품의 가격이 이전 수집 데이터보다 '하락'하는 추세를 감지하면 경보를 트리거(Trigger)합니다.
*   **직관적인 상세 정보 요약 발송**
    *   이메일을 열면 [어떤 항목]이 [어제 얼마]에서 -> [오늘 얼마]로 총 [얼마의 가격 방어선이 붕괴되었는지]가 아주 직관적이고 명료한 텍스트로 요약되어 도착합니다.
*   **유관 부서 통합 라우팅 (다수 메일링)**
    *   시스템 설정 시 단일 팀 단위의 공용 메일 혹은 유관 부서원들의 이메일을 `EMAIL_TO` 항목에 복수로 입력해둘 수 있습니다. 파편화 없는 빠른 정보 공유 및 보고 체계 확립이 가능해집니다.

---

## 3. 총평 및 기대 효과

*   **업무 시간 리소스 100% 절감:** 담당자가 매일 오전, 점심마다 루틴하게 반복 수행하던 수동 검색 및 엑셀 타이핑 취합 업무를 완전히 제거할 수 있습니다.
*   **견고한 데이터 기반 의사결정(Data-Driven):** 사람의 망각이나 연차 결원 등으로 인한 데이터 누락이 0으로 수렴하게 되며, 지속 축적된 클라우드 데이터를 통해 시장 가격 방어 정책 및 프로모션 타이밍을 가장 과학적으로 도출할 수 있습니다.
